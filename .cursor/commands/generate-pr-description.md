# PR説明文の生成

このコマンドは、現在のブランチとベースブランチ（デフォルト: develop）の差分を分析して、PR説明文を自動生成し、チャット上に表示します。

## 実行方法

このコマンドを実行すると、AIが以下の処理を自動的に行い、結果をチャット上に表示します：

1. 現在のブランチ名を取得
2. ベースブランチ（デフォルト: develop）との差分を分析
3. 変更されたファイルを分類（Backend/Frontend/その他）
4. コミット履歴を集約
5. PR説明文を生成してチャット上に表示

## 生成される内容

簡潔なPR説明文を生成します。以下の情報を含みます：

- **概要**: ブランチ情報と変更の要約
- **変更統計**: コミット数、変更ファイル数（Backend/Frontend別）
- **主なコミット**: 重要なコミットメッセージ（最大10個）
- **チェックリスト**: 基本的な確認項目

**注意**: 「PR説明」という見出しは含めません。直接内容から始まります。

生成されたPR説明文はチャット上に表示され、そのままコピー&ペーストしてGitHubのPR説明欄に使用できます。

## カスタマイズ

ベースブランチを指定したい場合は、コマンド実行時に「ベースブランチをmainにして」などと指定してください。
